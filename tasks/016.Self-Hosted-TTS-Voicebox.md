# Task 016 — Self-Hosted TTS Migration (Voicebox)

Objective: Add a self-hosted TTS backend based on `jamiepine/voicebox` so the
renderer can synthesize voice-over without ElevenLabs and support cloned voice
workflows on a home server.

Background
- Current production path uses ElevenLabs in `services/renderer/tts.py`.
- This task adds Voicebox as a supported backend, not a browser-side feature.
- Target repo: `https://github.com/jamiepine/voicebox`

Environment
- TTS_PROVIDER=elevenlabs|voicebox
- VOICEBOX_BASE_URL=
- VOICEBOX_API_KEY=                    # optional, only if the deployment enables auth
- VOICEBOX_VOICE_ID=                   # selected saved voice / speaker profile
- VOICEBOX_TIMEOUT_SEC=60

Implementation Steps
- [ ] Inspect Voicebox API surface and confirm synchronous audio generation for server-side rendering.
- [ ] Add provider abstraction in `services/renderer/tts.py` or split providers into `tts_elevenlabs.py` and `tts_voicebox.py`.
- [ ] Preserve deterministic cache keys across providers, including provider name and selected voice id.
- [ ] Support Voicebox-generated `vo.wav` output at renderer-compatible PCM settings.
- [ ] Add startup validation for `TTS_PROVIDER=voicebox` and required Voicebox env vars.
- [ ] Keep ElevenLabs as a supported fallback until Voicebox is proven stable.
- [ ] Document how to create or import a cloned voice profile for local use.
- [ ] Update Docker Compose or ops docs for running Voicebox alongside the stack on a home server.

Tests
- [ ] Unit: provider selection routes requests to Voicebox when configured.
- [ ] Unit: cache key changes when provider or voice profile changes.
- [ ] Integration (mock): Voicebox endpoint returns audio, renderer writes `vo.wav`, and cache reuse works on the second run.
- [ ] Failure: Voicebox timeout/5xx marks the render job `errored` with actionable logs.

Acceptance Criteria
- [ ] Renderer can complete a render job using `TTS_PROVIDER=voicebox` without ElevenLabs credentials.
- [ ] Voicebox output is compatible with subtitle generation and final ffmpeg muxing.
- [ ] No provider secrets are exposed to the browser.
- [ ] Operator docs explain how to switch between ElevenLabs and Voicebox.
